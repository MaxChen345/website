## 🧩 一、HTML 文件基本架構
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>我的首頁</title>
</head>
<body>
  <h1>標題</h1>
  <p>內容</p>
</body>
</html>

說明：
<!DOCTYPE html>：宣告 HTML5 文件格式，告訴瀏覽器使用標準模式。
<html>：整個網頁的根元素。
lang="zh-Hant"：設定語言為繁體中文。
<head>：放設定、標題、樣式等。
<meta charset="UTF-8">：設定文字編碼，避免亂碼。
<title>：瀏覽器分頁標題。
<body>：放實際內容（網頁主體）。



## 🧾 二、文字與段落標籤
<h1>～<h6> 標題</h1>
<p>段落文字</p>
<br> 換行
<hr> 水平線
說明：
<h1>～<h6>：標題層級（由大至小）。
<p>：建立段落。
<br>：強制換行。
<hr>：插入水平線。



✍️ 三、文字修飾標籤
<b>粗體</b>
<i>斜體</i>
<u>底線</u>
<mark>螢光標示</mark>
<small>小字</small>
<sub>下標</sub>
<sup>上標</sup>
<del>刪除線</del>
<em>語意強調</em>
<strong>語意加重</strong>
說明：
<b>、<strong>：粗體。
<i>、<em>：斜體與語意強調。
<mark>：像螢光筆一樣標示重點。
<sub> / <sup>：數學或化學公式用下標、上標。
<del>：刪除線。



🖼️ 四、圖片與超連結
<img src="images/me.jpg" alt="我的照片" width="200">
<a href="https://www.fju.edu.tw" target="_blank" title="前往輔大首頁">輔仁大學</a>
說明：
<img>：插入圖片。
src：圖片路徑。
alt：替代文字。
width / height：控制大小。
<a>：建立超連結。
href：目標網址。
target="_blank"：新分頁開啟。
title：滑鼠提示文字。



📋 五、清單（Lists）
<ul>
  <li>項目1</li>
  <li>項目2</li>
</ul>

<ol>
  <li>第一步</li>
  <li>第二步</li>
</ol>

<dl>
  <dt>HTML</dt>
  <dd>超文件標示語言</dd>
</dl>
說明：
<ul>：無序清單（●）。
<ol>：有序清單（1, 2, 3）。
<li>：清單項目。
<dl>：定義清單（詞條 + 說明）。
<dt>：名詞。
<dd>：解釋。



📊 六、表格（Table）
<table border="1">
  <tr>
    <th>姓名</th>
    <th>學號</th>
  </tr>
  <tr>
    <td>陳泰丞</td>
    <td>1414421256</td>
  </tr>
</table>
說明：
<table>：建立表格。
<tr>：表格列（table row）。
<th>：表頭（粗體、置中）。
<td>：資料儲存格。

常見屬性：
border：邊框線。
cellpadding：內距。
cellspacing：間距。



🧱 七、排版與分區
<div>主要區塊</div>
<span>行內文字</span>
<pre>保留空白與換行</pre>
<blockquote>引用段落</blockquote>
說明：
<div>：區塊容器，用於版面分組。
<span>：行內區塊，用於特定字詞。
<pre>：保留排版格式。
<blockquote>：引用文字。



🧠 八、HTML5 語意化標籤
<header>頁首</header>
<nav>導覽列</nav>
<main>主要內容</main>
<section>章節區塊</section>
<article>獨立文章</article>
<aside>側邊補充</aside>
<footer>頁尾</footer>

說明：
標籤	       功能	    範例用途
<header>	  頁首	  放 logo、標題
<nav>	     導覽列	     主選單
<main>	     主內容	  頁面主要內容（限一個）
<section>   主題區塊	分章節用
<article>	獨立內容	文章、貼文
<aside>	    側邊區塊	補充資訊
<footer>	  頁尾	  版權、聯絡方式



🧩 九、其他實用標籤
<time datetime="2025-10-28">2025/10/28</time>
<abbr title="HyperText Markup Language">HTML</abbr>
<code>console.log("Hello");</code>
<address>輔仁大學 統計資訊學系</address>
說明：
<time>：表示時間或日期。
<abbr>：縮寫詞 + 全名提示。
<code>：程式碼片段。
<address>：聯絡資訊。



🎨 十、CSS 語法與屬性
選擇器 {
  屬性: 值;
}

常見屬性表：
屬性	                功能	範例
color	            文字顏色	color: red;
background-color	背景色	    background-color: #f5f5f5;
font-size	        字體大小	font-size: 18px;
font-family	        字體樣式	font-family: 微軟正黑體;
text-align	        對齊	   text-align: center;
margin	            外距	   margin-top: 20px;
padding	            內距	   padding: 10px;
border	            邊框	   border: 1px solid #000;
width / height	    尺寸	   width: 200px;
list-style-type	    清單符號	list-style-type: square;



🧮 十一、綜合 HTML5 範例
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>HTML5 結構範例</title>
  <style>
    body {font-family: 微軟正黑體; background-color: #f5f5f5; text-align: center;}
    img {width: 200px; border-radius: 10px; margin-top: 10px;}
  </style>
</head>
<body>
  <header>
    <h1>陳泰丞的首頁</h1>
    <p>統計資訊學系一年級</p>
    <p>最後更新：<time>2025/10/28</time></p>
  </header>

  <main>
    <p>照片：</p>
    <img src="images/me.jpg" alt="我的照片">
  </main>

  <footer>
    <p>本網站使用 HTML5 / CSS3 製作。</p>
  </footer>
</body>
</html>



⚠️ 十二、常見錯誤 vs 正確寫法
錯誤語法	                問題	                正確寫法
<div><h1></div>	        標籤未正確閉合	              <div><h1></h1></div>
<p><div>文字</div></p>	 區塊元素不能放在 <p> 裡	   <div><p>文字</p></div>
<img>                   沒有 alt	無障礙不友善	  <img src="a.jpg" alt="說明文字">
<section>               無 <h2> 標題	語意失效	  <section><h2>主題</h2>內容</section>
全用 <div>	             無語意	                     改用 <header>、<main>、<footer>



📘 十三、HTML5 語意化的優點
1️⃣ 幫搜尋引擎理解頁面結構（SEO）
2️⃣ 提升無障礙與閱讀器可用性
3️⃣ 提高可維護性與團隊協作可讀性
4️⃣ 工具能自動判斷錯誤（Lighthouse、axe）
5️⃣ 結構穩定、樣式更易控管
6️⃣ 為未來轉換（APP、RSS、AI 摘要）打好基礎

💡 語意化不是為了變漂亮，而是讓「人、機器、與未來的你」都能理解內容。
